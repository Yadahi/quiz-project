## Notes:

### Beginning the assignment:

1) Clone this assignment to a folder on your external harddrive, OneDrive, or other convenient location.
2) Open Visual Studio Code and close any currently open projects.
3) Open the cloned folder.   
4) Open the 'Assignment5.ipynb' file to begin the assignment.
5) Use 'quiz.ts' to complete Part C.
6) Run unit tests located in 'quiz_test.ts' to help complete the program.

### Submitting the assignment:

1) Complete all the questions in the 'Assignment5.ipynb' file.
2) Save all files.
3) Create a PDF of the completed Jupyter Notebook and the TypeScript file.
4) Commit and push all files to GitHub.
5) Upload all the PDFs to the Assignment 5 dropbox in Brightspace.
6) That's it!
